#### Git commit template


>>>
    
    <type>(scope): <subject>
	
	<body>

	<footer>

	1. type field

	    - feat: new feature

		- fix: fix bug

		- docs: documentation

		- style: code format

		- refactor: refactor code

		- test: add testing

		- chore: build or helper tools
	
	2. scope

	    - commit scope

	3. subject

	    - description

	4. body
	    
	    - change log content

	5. footer
	   
	    - action
